1.3 ATA

Modèle Allongé

Chambre souple confortable, idéale pour une utilisation en position allongée.

  • Pression : 1.3 ATA / 4 psi
  • Modèles : ST701, ST801, ST901
  • Matière : TPU
  • Grandes fenêtres transparentes
  • Concentrateur 5 L/min
Assis

Modèle Assis

Conçu pour les patients qui préfèrent rester assis ou semi-couchés.

  • Pression : 1.3 ATA / 4 psi
  • Modèle : ST1700
  • Utilisation simple
  • Chaise pliante en option
  • Confort anti-claustrophobie
1.5 ATA

Hard Type

Chambre rigide professionnelle avec système machine tout-en-un.

  • Pression : 1.5 ATA / 7 psi
  • Modèles : HP1501-75 / HP1501-90
  • Acier inoxydable + polycarbonate
  • Porte coulissante
  • Système interphone
Pet Care

Chambre pour animaux

Solution dédiée aux soins vétérinaires et à la récupération animale.

  • Modèles : HP600 / HP700
  • Pression : 1.3 à 1.5 ATA
  • Aluminium / acrylique / acier inox
  • Cabine adaptée aux animaux
  • Machine système intégrée
